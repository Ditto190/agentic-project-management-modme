# Proposal & RFP Input

## Description

Develop and contribute to client proposals, RFP responses, and pre-sales artefacts.

## Deliverables

- [[DLV-011]] Delivery Approach Narrative
- [[DLV-014]] Assumptions Log

## Key Questions

1. Does the delivery approach in the proposal match what we can actually deliver?
2. Have all commercial and technical assumptions been captured?
3. Is the SoW aligned with the proposal narrative?

- Proposal, RFP Response
