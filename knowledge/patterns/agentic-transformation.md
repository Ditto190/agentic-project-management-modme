# Agentic Transformation Pattern

## Overview

Agentic transformation engagements introduce AI agents into client business processes. They require careful scope definition, change management, and governance.

## Delivery Adaptations

| Standard Activity | Agentic Adaptation |
|------------------|-------------------|
| Scope Definition | Include AI ethics and governance scope; define human-in-the-loop requirements |
| RAID Management | Add AI-specific risks: hallucination, bias, data quality, adoption resistance |
| Estimation | Include model training, evaluation, and iteration cycles |
| Governance | Add AI governance layer: model cards, bias audits, usage policies |
